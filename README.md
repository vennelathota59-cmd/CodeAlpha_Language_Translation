# 🌍 AI Language Translation App

## 📌 Project Overview

The AI Language Translation App is a simple and user-friendly web application developed using Python and Streamlit. It allows users to translate text into multiple languages instantly using the Google Translator API.

This project was developed as part of the CodeAlpha AI Internship.

---

## ✨ Features

- 🌍 Supports 25+ Languages
- ⚡ Fast and Accurate Translation
- 📊 Character Counter
- 📝 Word Counter
- 📥 Download Translated Text
- 🗑️ Clear Button
- 🛡️ Error Handling
- 🎈 Success Animation
- 💻 Clean and Responsive User Interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- deep-translator

## ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/vennelathota59-cmd/CodeAlpha_Language_Translation.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

---

## 📸 Output

The application translates text instantly into the selected language and allows users to download the translated text.
